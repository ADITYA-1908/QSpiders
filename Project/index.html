comparison between JavaScript and Java in 10 sentences:

Java is a statically typed, object-oriented programming language used mainly for building standalone applications. JavaScript is a dynamically typed, interpreted scripting language primarily used for web development. Java code needs to be compiled using a compiler, while JavaScript runs directly in web browsers. Java is commonly used in Android apps, enterprise applications, and backend systems. JavaScript is mostly used for adding interactivity to websites (like buttons, animations, etc.). Java runs on the Java Virtual Machine (JVM), while JavaScript runs in a browser or on Node.js. Java is class-based and supports multithreading and concurrency. JavaScript is prototype-based and single-threaded (but supports async behavior using events/promises). Java uses strong type-checking, whereas JavaScript allows more flexibility with variable types. Despite similar names, Java and JavaScript are completely different languages with different use cases.
